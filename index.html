About The Project

This project is a JavaScrpt powered responsive resume website alone with HTML and CSS. The objective was to develop a pixel-perfect responsive resume website that will display personal information, images, descriptions and projects with links. The style and layout were structured to be the same as the provided mockup. Additional features and tools used include Google Map's API, jQuery, Bootstrap and animation libraries.
